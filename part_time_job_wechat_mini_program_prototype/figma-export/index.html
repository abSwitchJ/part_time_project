下一步 · 进 Figma

  1. 下载 ZIP 解压(右上角的下载卡片)
  2. 打开 Figma,新建文件,直接把 svg/ 里的 7 个文件拖进画布
  3. 选中导入的 group,Cmd+Shift+G 一层一层 Ungroup,文本节点可双击编辑
  4. 字体需要本地装好 DM Serif Display / DM Serif Text / EB Garamond / Playfair Display / JetBrains Mono / Noto Serif SC,否则会回退
